我是...

高逸扬

此实时留言板系统的作者


此系统使用了 Node.JS, MySQL, JavaScript 作为主要技术栈；网络通信方面主要使用了 socket.io，底层使用了 WebSocket，服务器托管在 DigitalOcean 新加坡机房并实现了双机热切换的灾备（通过 MySQL 主从库实现）。

从一开始，除了使用了公共依赖库之外，其余代码便是均由高逸扬同学我一人独立完成。我是一位 16 岁的勤礼外语中学学生，热爱编程、摄影、航拍与影片制作。我现在所掌握的语言大致有 PHP, SQL, JavaScript (ES6), HTML5, CSS3, Python. 在毕业之前，我还计划学习 Ruby on Rails, C, Objective-C, Java 和 Android 编程。另外，我还参与制作了大量的学校宣传视频和纪念视频，例如最近一次发布在学校公众号的勤礼外语中学 2018 运动会短片(视频大小 90MB；点击链接即可观看)，即是主要使用了 Adobe Premiere Pro，Adobe After Effects，和 Adobe Audition 这三款专业级的非线性剪辑软件进行制作的。

我希望使用代码与程序来改变人们生活的方式，并通过这些方式来对人类发展产生正面的影响。
